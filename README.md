### About the Project
* **Topic:** [The Kochi Accessibility Compass is an interactive visualisation designed to measure and showcase the accessibility of essential urban services across Kochi. Developed through extensive fieldwork and data collection, the dashboard transforms raw ward-level information into meaningful accessibility scores. These scores capture how easily residents can access key facilities and opportunities within their neighbourhoods.
